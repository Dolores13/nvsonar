# Contributing to NVSonar

Thank you for considering contributing to NVSonar!  
This guide will help you set up a development environment and contribute smoothly.

---

## 1. Fork and Clone

1. Fork the repository on GitHub using the **Fork** button in the top-right corner.  
2. Clone your fork locally:

```bash
git clone https://github.com/<YOUR-USERNAME>/nvsonar.git
cd nvsonar
```

## 2. Install Package
Install the package from the local workspace in editable mode
```bash
pip install -e .
```

## 3. Run
```bash
nvsonar
```
